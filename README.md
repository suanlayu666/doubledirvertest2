🏎️ STM32 智能小车控制系统
一个基于 STM32F103 的多模式智能小车项目，实现了温控调速、本地控制、蓝牙遥控以及自动避障功能。

🌟 项目亮点
   多模式状态机设计：系统状态在 AUTO、HAND、BLUE 三种模式间灵活切换。
   温控巡航算法：利用单总线 DS18B20 实时测温，根据温度梯度动态调整 PWM 占空比（speed）。
   自适应避障系统：调用 TIM 捕获超声波 Echo 信号，一旦距离 $d \le 25cm$，强制接管电机控制权执行快速转向。
   
   人机交互增强：
   本地控制：通过 GPIO 扫描 GPIOB 的引脚（按键 0 和 10）实现速度的步进加减（±5%)。
   远程控制：使用 USART 监听蓝牙指令（如 F/B/L/R/S, +/-），实现无线操控。
   实时反馈：优化后的 OLED 驱动（利用 sprintf），动态刷新 PWM 占比、温度、方向（FWD/REV）及距离。

算法逻辑核心解析
主循环（Main Loop）：根据变量 mode 在 0 (AUTO)、1 (HAND)、2 (BLUE) 三个不同的逻辑块之间切换。
模式2：蓝牙遥控 + 自动避障：优先级逻辑：首先通过蓝牙串口中断更新 bt_turn。
避障接管算法：当超声波测距距离 $d \le 25cm$ 时，强制将局部变量 final_turn 设为 2（右转），覆盖蓝牙指令，直至距离安全。
电机驱动执行算法：根据 direction 和 final_turn（或 bt_turn），配置 GPIOA 5, 6, 11, 12 的高低电平及两个 PWM 通道的 CCR 值，实现前进、后退、差速转向。

🚀 如何运行
环境配置：你需要 Keil MDK v5 及 STM32CubeF1 库。
连接硬件：按照“硬件清单”部分的引脚定义连接小车外设。
烧录代码：打开工程文件，编译并烧录至 STM32F103C8T6。

操作小车：
AUTO 模式 (默认)：小车将根据环境温度自动调整前进速度。
HAND 模式：按下一侧按键加速，另一侧按键减速。
BLUE 模式：连接手机蓝牙APP，发送 'F' 'B' 'L' 'R' 'S' '+' '-' 控制小车，同时小车会自动避障。
